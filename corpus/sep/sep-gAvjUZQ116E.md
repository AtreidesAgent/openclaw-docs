## Analysis

### Summary
Dylan speaks with Al Morris, CEO and founder of Koii Network, which aims to democratize the internet by rewarding users for contributing computing resources and attention. The conversation covers Koii's mission to build a truly decentralized internet, how users can earn tokens by running tasks on their devices, the various iterations of the Koii Network, an IBM partnership for compute infrastructure, and the vision for a user-owned attention economy.

### Topics Covered
- Koii Network's mission: abstracting incentives from intermediaries via open-source distributed architecture
- User data ownership and control: empowering users rather than platforms
- The attention economy: rewarding users for contributing resources and attention
- Koii tasks: community-run compute tasks that reward participants in various tokens
- K2 settlement layer for transaction processing and consensus
- IBM partnership for server infrastructure to support larger compute tasks
- The "Uber driver" analogy: matching task requirements with user hardware capabilities
- Progression from desktop nodes to mobile device participation
- The Koii plant "Oliver" as a community engagement experiment
- Open-source philosophy and community-driven development
- Multiple token incentive structures: Koii tokens for settlement, custom tokens for specific tasks

### Dylan's Takes & Opinions
- Dylan opened by asking about the "Koii plant status" (Oliver), showing a personal rapport and recurring inside joke with Al
- He expressed excitement about the decentralized internet thesis and user data ownership
- Dylan was interested in the growing scale of entities needing decentralized compute, from individual users to enterprise-level IBM partnerships

### Guest Perspectives
- **Al Morris** (Founder & CEO, Koii Network): Envisions a consumer-controlled community network rather than one owned by Facebook or similar companies. Described the Uber-like model of matching compute tasks with available user hardware. Emphasized that Koii tokens are only used for the settlement/consensus layer, while individual tasks can reward in any token. Noted the transition from purely crypto-native users to broader technology enthusiasts interested in decentralized computing. Highlighted that the final tech docs were recently released, opening the network to broader participation.

### People & Projects Mentioned

- **IBM** - Partnership for server infrastructure
- **Oliver** - The Koii plant; a pathos leaf growing as a community engagement piece
