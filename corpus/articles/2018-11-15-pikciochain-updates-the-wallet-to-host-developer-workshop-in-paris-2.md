# PikcioChain updates The Wallet, to host developer workshop in Paris

**Published:** Thu, 15 Nov 2018 16:32:03 +0000
**URL:** https://neonewstoday.com/general/pikciochian-updates-the-wallet-to-host-developer-workshop-in-paris/
**Tags:** General, PKC

---

PikcioChain has announced an update to its mobile wallet for Android devices, and has an upcoming developers workshop in Paris, France. The Wallet updates include functionality with regards to token sale and identity features. PikcioChain aims to let businesses exchange personal data with security and confidence, whilst giving individuals complete control of their personal data and enabling them to realize its financial value.

The November wallet update includes:

- New token listings,

- Ability to create and use NEO and Ethereum wallets for transactions,

- Private keys for imported wallets can be saved and used for transactions,

- User account information will be automatically populated when using ID and proof of address as initial data, and

- Phone number validation via SMS messaging.

PikcioChain Developer Workshop in Paris

On Thursday, November 29th PikcioChain will be hosting a Blockchain Architecture and PikcioChain Discovery workshop in Paris, France. 

The event will begin with a general overview of PikcioChain, then teach individuals about the structure of the network’s nodes and using APIs and SDKs to access and utilize the network.

The speaker for the three-hour workshop will be Jorick Lartigau, co-founder and lab coordinator for Pikcio AG, PikcioChain’s parent company.
