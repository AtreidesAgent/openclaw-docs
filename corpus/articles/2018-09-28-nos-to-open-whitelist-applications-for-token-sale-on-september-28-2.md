# nOS to open whitelist applications for token sale on September 28

**Published:** Fri, 28 Sep 2018 04:11:39 +0000
**URL:** https://neonewstoday.com/general/nos-to-open-whitelist-applications-for-token-sale-on-september-28/
**Tags:** General, NOS

---

The nOS token sale whitelist applications will open at 20:00 (GMT +2) on Friday, September 28th, 2018. nOS is a virtual operating system that will run on mobile and desktop, and function like a browser and app store. The token sale application period will end at 20:00 (GMT +2) on Friday, October 5th, 2018.

The application process will be conducted through a lottery system, where 15,000 public sale applicants will be randomly selected to participate. All pre-sale applicants are automatically provided with guaranteed access to the public token sale, and will be able to perform the know your customer (KYC) requirements during the lottery application process.

Participants who have been randomly selected in the lottery will need to provide limited KYC details (name, country, identity document number). Once KYC information has been registered, the details will be unalterable. If incorrect or falsified information has been submitted, the participant will be disqualified from partaking in the public token sale.

A “fair selection procedure” will be in place to guarantee the randomness of individuals selected to participate in the public token sale. nOS will verify its fair selection by sharing the “MD5 checksum” of the text file, which can’t be altered once the lottery has begun. The MD5 checksum is “a unique hashed version of specific data. If the data (in this case a text file) is shared later, anyone can retrieve the same MD5 hash from it, verifying that the file is legitimate.” 

To participate, interested parties will need to download the latest version of the nOS Client, and create an account at nOS.app. The nOS.app account can be created using the nOS Client which can be downloaded at the following link:

https://github.com/nos/client/releases 

Individuals who opt to contribute Ethereum (ETH) in the public token sale will be able to do so by clicking their “username on the top right, going to Settings, and pasting [their] Ethereum address.”

Note: Citizens of China, Ethiopia, Iran, North Korea (DPRK), Sri Lanka, Sudan, Syria, United States Minor Outlying Islands, United States of America, Virgin Islands (U.S.), Yemen and any other countries where the participation in Virtual Token Generation Events or Utility Token Generation Events is forbidden, are restricted from participating.
