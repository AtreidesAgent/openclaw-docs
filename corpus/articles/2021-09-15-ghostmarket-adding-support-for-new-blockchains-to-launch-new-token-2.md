# GhostMarket adding support for new blockchains, to launch new token

**Published:** Wed, 15 Sep 2021 22:30:46 +0000
**URL:** https://neonewstoday.com/general/ghostmarket-adding-support-for-new-blockchains-to-launch-new-token/
**Tags:** Exchanges, General, NFT, GhostMarket, GM

---

GhostMarket has announced upcoming support for two new blockchains and the pending launch of a governance token that will live on each chain. GhostMarket founder, Vincent Geneste, participated in the project’s first AMA on Discord to discuss progress and next steps.

Over the next six weeks, multi-chain NFT marketplace and explorer GhostMarket will add support for the Ethereum and Avalanche blockchains. Geneste also noted that the development team will examine adding the Tezos, Wax, Solana, and Flow blockchains to the platform.

According to Geneste, the team will launch a utility token, GM, which will be used for rewards and other purposes. GhostMarket users will be able to use GM to pay for fees such as minting or trading NFTs. GM will also be used for governance purposes, like voting on new tokens that can be used to trade NFTs. Other use cases might include using GM to promote an NFT or access to special artist drops based on GM holdings.

The GM token will live as the fungible token standard on the various blockchains supported by GhostMarket (i.e., NEP-17, ERC-20). GhostMarket will also host a swapping interface on the platform, so users can seamlessly swap GM between different chains.

A portion of the proceeds from the GM token sale will reward early GhostMarket platform users who used the platform. Geneste said a pool equivalent to US $100,000 has been set aside for this purpose.

Looking forward, more information about the token sale is expected to be shared in the upcoming weeks. The GhostMarket team aims to expand and is searching for a senior front-end developer, a system operations engineer, and a mobile application engineer.

Currently, GhostMarket is a multi-chain platform. It doesn’t support the minting of NFTs on one chain and the migration of that NFT to another chain. Geneste noted that this cross-chain functionality will be coming to GhostMarket in the next eight weeks.
