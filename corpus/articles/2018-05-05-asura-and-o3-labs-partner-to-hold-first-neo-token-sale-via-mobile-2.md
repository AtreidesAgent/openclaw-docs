# Asura and O3 Labs partner to hold first NEO token sale via mobile

**Published:** Sat, 05 May 2018 17:43:31 +0000
**URL:** https://neonewstoday.com/general/asura-and-o3-partner-for-asa-token-sale/
**Tags:** General, O3

---

On May 4, 2018, Asura Coin announced a partnership with O3 Labs for Asura’s upcoming public token sale. Asura Coin – an e-sports gaming platform for betting, tournaments, and community matchmaking – will be the first public token sale to utilize the O3 mobile wallet application. O3 is currently the only mobile wallet that supports NEP-5 tokens.

O3 Labs recently released the below gif which simulates what the user experience will be like as they participate in a token sale. O3 is available for both iOS and Android users. The mobile wallet does not store an individuals private keys, and provides token balance and performance tracking.

The O3 Mobile Wallet can be downloaded for iOS here, and for Android here.

Asura Token Sale

The Asura Coin (ASA) token sale is planned to begin on May 21, 2018, and will either end on June 15th or whenever the hard cap is reached. The Know Your Customer (KYC)/whitelist registration is currently underway, and can be accessed here.

The NEO address used to register for the KYC/whitelist will be the same address participants will need to contribute from during the crowdsale. Additionally, NEO will be the only currency accepted for the public token sale.

650 million ASA tokens (65% of the total supply) will be available for purchase during the token sale, priced at $0.02 USD/token. A hardcap of $12,000,000 USD and a soft cap of $2,500,000 USD has been set.
