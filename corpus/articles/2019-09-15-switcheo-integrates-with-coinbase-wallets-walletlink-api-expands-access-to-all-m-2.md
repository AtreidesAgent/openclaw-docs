# Switcheo integrates with Coinbase Wallet’s WalletLink API, expands access to all modern browsers

**Published:** Sun, 15 Sep 2019 14:58:31 +0000
**URL:** https://neonewstoday.com/general/switcheo-integrates-with-coinbase-wallets-walletlink-api-expands-access-to-all-modern-browsers/
**Tags:** Exchanges, General, SWTH

---

Switcheo has announced it has integrated its non-custodial exchange with Coinbase Wallet’s WalletLink API. The WalletLink integration now allows “Coinbase Wallet users [to] securely access and administer their funds to trade on Switcheo from any modern browser.”

In December 2018, Switcheo announced its initial integration with Coinbase Wallet. Coinbase Wallet is a mobile application that allows users to manage ETH and ERC-20 tokens, store NFT assets, and access third party dApps through it’s dApp browser. The integration offered users the ability trade on Switcheo Exchange through the in-app mobile dApp browser.

The WalletLink integration extends this support by allowing Switcheo users to link their Coinbase Wallet to a desktop browser by scanning a QR code. This “creates a secure encrypted link” between the wallet app and the browser, and will push notifications to the users phone when a transaction needs to be signed.

In a conversation with NEO News Today, Switcheo CEO, Ivan Poon said:

WalletLink makes it even easier for users to trade from self-custody wallets like Coinbase Wallet and we’ll continue integrating with such products to improve the experience when using Switcheo!

The Coinbase Wallet is the second non-custodial wallet application to be integrated with Switcheo. In August 2018, Switcheo and also added support for users to trade on its exchange directly through the O3 Wallet. Switcheo Exchange was the first dApp to utilize the O3 dApp browser.
