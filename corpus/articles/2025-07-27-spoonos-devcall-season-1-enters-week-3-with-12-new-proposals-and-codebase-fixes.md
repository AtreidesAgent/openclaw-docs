# SpoonOS DevCall Season 1 enters Week 3 with 12 new proposals and codebase fixes

**Published:** Sun, 27 Jul 2025 23:49:40 +0000
**URL:** https://neonewstoday.com/ai/spoonos-devcall-season-1-enters-week-3-with-12-new-proposals-and-codebase-fixes/
**Tags:** AI, General, SpoonOS

---

As Season 1 of the SpoonOS Developer Call enters its final stretch, the community has seen consistent growth in pitches and proposals toward AI agent innovation across DeFi, prediction markets, and developer infrastructure.

DevCall Season 1 is a month-long initiative encouraging developers to prototype AI-powered agents using SpoonOS, an early-stage framework for building Web3-integrated, LLM-powered autonomous programs. The program seeks to gather developer feedback, foster community engagement, and identify key contributors for ongoing development.

This week, 12 proposals and patches to existing codebase were submitted for approval, showcasing novel use cases and critical core enhancements.

SpoonMiniHub

Team: Code Crafters (Om Baviskar, Harshal Suroshe)

SpoonMiniHub aims to be a decentralized launchpad for AI-powered minigames, integrating the Spoon Agent Kit for game moderation, feedback collection, and monetization. Built on Ethereum with Solidity, the platform supports independent developers in creating and testing Web3-native minigames.

SynergyX

Team: QuantumLayer (Kai, Minh)

SynergyX is an autonomous DeFi agent optimized for real-time execution of cross-chain yield and arbitrage strategies. It utilizes SpoonVM and reinforcement learning to rebalance user portfolios across EVM-compatible networks.

Neo DeFi Guardian Agent

Team: Web3Devz (Solo developer)

This agent seeks to monitor Neo blockchain smart contracts for common risk patterns using LLMs. It serves auditors, DeFi users, and DAOs by issuing threat alerts via CLI, dashboards, and Discord.

On-Chain AI Investment Agent

Team: Anonymous

This SpoonOS-based agent will be designed to scan DEX activity and token launches to generate investment signals. It publishes results to smart contracts and social channels, featuring a modular stack for AI evaluation and user notifications. Revenue models include API licensing, subscriptions, and white-label offerings.

On-Chain AI Investment Agent

Team: Natalie1t (Solo developer)

A second, distinct proposal with the same name, is a build that intends to run entirely on-chain, using OpenRouter APIs for signal generation and scoring. It is designed for funds, DAOs, and individual investors and offers monetization via NFT-gated access and subscriptions.

News ReAct Agent & Data MCP Server for Polymarket

Team: Zhi Wang

This dual-agent system will seek to be tailored for Polymarket prediction markets. It includes a News Insight server for sentiment scoring and data aggregation, and a React agent for narrative analysis and market reactivity. A freemium model is suggested for developer adoption.

Multiple CLI Issues in SpoonAI CLI

Team: Ghost (Sambit Sargam Ekalabya)

This patch proposes to resolve three SpoonAI CLI issues: duplicate methods, invalid input errors, and failed session resets. Fixes are consolidated in PR #49 with detailed testing instructions.

On-chain Data Analyst Agent for Custom Strategy Execution

Team: Team Douglas (Solo developer)

This SpoonOS React agent envisages enable custom trading strategy execution using on-chain data, Chainbase, and GeckoTerminal APIs. It includes a UI dashboard and is designed for retail DeFi traders.

TEE/HSM Secure Wallet Integration for SpoonOS

Team: TEE Fans (Solo developer)

This proposal proposes support for Intel SGX and HSM-based key management in SpoonOS. It targets enterprise security needs for autonomous agents, with potential licensing and compliance monetization models.

MCP Multi-Chain Expansion Proposals (#39–41)

Team: superxiao

A three-part proposal to enhance the MCP tool:

- #39: Enhance MCP Tool with Unified Multi-Chain Support and Developer-Friendly APIs

- #40: Add One-Stop On-Chain Transaction Module to MCP for Multi-Chain DeFi Operations

- #41: Integrate Cross-Chain Bridging Capabilities into MCP for Unified Multi-Asset Transfer Support

This three-part proposal aims to introduce a major upgrade to SpoonOS’s MCP tool, intended to support full-stack on-chain functionality across multiple blockchain networks. The enhancements include plug-and-play support for Solana, Ethereum, and Base; a unified transaction module for swaps, liquidity provisioning, and custom contract interactions; and integrated cross-chain bridging through protocols like Wormhole and LayerZero. Together, these upgrades seek to position MCP as a comprehensive multi-chain execution layer for autonomous agents.

DevCall Season 1 concludes next week. The final batch of proposals and closing reflections will be released by the SpoonOS team on Friday, Aug. 1.
