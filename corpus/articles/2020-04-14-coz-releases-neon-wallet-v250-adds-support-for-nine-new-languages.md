# COZ releases Neon Wallet v2.5.0, adds support for nine new languages

**Published:** Tue, 14 Apr 2020 21:44:45 +0000
**URL:** https://neonewstoday.com/general/coz-releases-neon-wallet-v2-5-0-adds-support-for-9-new-languages/
**Tags:** General, COZ, GAS, NEO

---

COZ has released v2.5.0 of Neon Wallet, which most notably integrates support for nine new languages. The latest release can be downloaded for Mac, Windows, and Linux here.

To offer greater access to the international Neo community, v2.5.0 features language support for Arabic, Chinese, French, German, Italian, Korean, Portuguese, Turkish, and Vietnamese.

Source: COZ

The internalization of the Neon Wallet was a significant undertaking that updated approximately 13,000 lines of code and more than 160 files. Through these internalization upgrades, Neon Wallet’s underlying infrastructure is claimed to support the addition of new languages simply and efficiently.

In a conversation with Neo News Today, Neon Wallet maintainer, Max Lasky (@comountainclimber), said:

We intend to make Neon the go-to wallet in the Neo ecosystem. Because Neo is such an international project, we thought that this would be a valuable use of our time and resources. Now that these pieces are in place, we can easily add new translations and develop more technical features that have instructions/details in every language.

Developers can now add support for new languages into the open-source wallet by adding a JSON file with the appropriate translations here.

Other changes to the Neon Wallet app include improvements to dependencies and automated test coverage, the removal of unused dependencies, and CSS and asset updates.
