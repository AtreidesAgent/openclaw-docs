# NEO-ONE workshop on building NEO tokens – January 15th – Seattle

**Published:** Fri, 04 Jan 2019 22:11:39 +0000
**URL:** https://neonewstoday.com/events/neo-one-workshop-on-building-neo-tokens-january-15th-seattle/
**Tags:** Events, NEO-ONE

---

NEO Blockchain Seattle will be hosting a workshop where participants will be able to learn how to use NEO-ONE to build a token on the NEO blockchain. NEO-ONE is an all-in-one dApp development suite that was launched in November of 2018, covered by NEO News Today.

Date and Location

The event will take place from 6:30 pm to 8:30 pm on Tuesday, January 15th, 2019.

The workshop is located at:

Galvanize

111 South Jackson Street

Seattle, WA

Agenda

- 6:30 pm – Arrival

- 6:45 pm – Demo NEO-ONE Lesson 1: How to Build a Token

- 7:30 pm – Question and Answer session

- 7:45 pm – Free work time for attendees to continue building token or try new ideas

NEO-ONE developers will be on hand to provide support to participants.

It is highly recommended event attendees bring their own computer and review the “Environment Setup” page to get oriented with the platform. The Environment Setup page can be found at the following link:

https://neo-one.io/docs/environment-setup/

Should a participant run into difficulties when setting up the environment, they’ll be able to query the developers on hand at the workshop.

Registration

There is no cost to attend the event.

To register for the event, visit the NEO Blockchain Seattle meetup link below:

https://www.meetup.com/meetup-group-FMlwozoX/events/257354856/
